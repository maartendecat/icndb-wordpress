== Description ==

This plugin adds a small Wordpress widget that shows a random personalized Chuck Norris joke on your blog, starring yourself.
These jokes are taken from a database on icndb.com.
For regular Chuck Norris jokes, please refer to the [Chuck Norris Jokes Widget](http://wordpress.org/extend/plugins/chuck-norris-joke-widget/ "Chuck Norris Jokes Widget").

== Installation ==

To install this plugin:

1. Download the plugin and extract the contents.
1. Upload the entire plugin directory to '/wp-content/plugins/'.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
1. Drag the "Personalized Chuck Norris Joke" widget in a widget area through the 'Appearance – Widgets' menu.
1. Adjust the name of the main character in the joke or keep the default settings. Then press save and close.

You should now see a different Chuck Norris joke on your blog every time the page loads.
